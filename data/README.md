# Data

- `energy_efficiency.csv` — датасет Energy Efficiency (UCI/ENB2012), 8 входов + 2 выхода.
- `energy_efficiency_raw.csv` — исходный CSV с оригинальными колонками `X1..X8,Y1,Y2`.

Источник:
- https://archive.ics.uci.edu/dataset/242/energy+efficiency
- Зеркало CSV: https://raw.githubusercontent.com/DarrenCook/h2o/bk/datasets/ENB2012_data.csv
